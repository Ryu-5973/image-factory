# image-factory

`image-factory` 的初始工程骨架。

当前仓库先保持技术栈中立，提供最基础的目录结构与项目说明，方便后续按实际方案扩展为图像处理服务、桌面工具、Web 应用或脚本集合。

## 目录结构

```text
.
|-- assets/        # 静态资源、样例图片、参考素材
|-- docs/          # 设计文档、接口说明、方案记录
|-- scripts/       # 构建、开发、数据处理脚本
|-- src/           # 核心源码
`-- tests/         # 测试代码
```
